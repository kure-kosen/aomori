## Install

### OpenCV

```
$ brew tap homebrew/science
$ brew install opencv3 --with-python3
$ brew link opencv3 --force
```

### python3

```
$ brew install python3
$ pip install virtualenv
$ virtualenv aomori_env -p python3
$ pip install -r requirement.txt
```
